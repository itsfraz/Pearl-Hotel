## How to Take Screenshots

### Option 1: Browser DevTools (Recommended)

1. Open the page you want to capture
2. Press `F12` to open DevTools
3. Press `Ctrl + Shift + P` (Windows) or `Cmd + Shift + P` (Mac)
4. Type "Capture full size screenshot"
5. Save to `screenshots/` folder

### Option 2: Windows Snipping Tool

1. Press `Windows + Shift + S`
2. Select area to capture
3. Save to `screenshots/` folder

### Option 3: Mac Screenshot

1. Press `Cmd + Shift + 4`
2. Select area to capture
3. Save to `screenshots/` folder

## Tips for Great Screenshots

1. **Clean Browser:** Close unnecessary tabs, hide bookmarks bar
2. **Full Screen:** Use F11 for full-screen mode
3. **Zoom:** Set browser zoom to 100%
4. **Test Data:** Use realistic test data (not "test test test")
5. **Highlight Features:** Show the amenities dropdown in admin-rooms.png!
6. **Consistency:** Use the same browser for all screenshots
7. **Quality:** Use PNG format for better quality
